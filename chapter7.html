1. Searching File Content: grep / egrep / fgrep

The grep command searches files for lines that match a given pattern. It supports various options and regular expressions.

Common options:

  • -i: Ignore case
  • -l: Print only file names with matching lines
  • -n: Show line numbers
  • -v: Invert match (exclude matching lines)
  • -c: Count matching lines
  • -w: Match whole words

Using grep Examples

  • grep -i unix grep.dat: Case-insensitive match
  • grep -l unix grep.dat: Show file names with matches
  • grep -n unix grep.dat: Show line numbers
  • grep -v unix grep.dat: Show lines without the pattern
  • grep -c 123 grep.dat: Count lines with "123"
  • grep -w unix grep.dat: Match the whole word "unix"

Regular Expressions with grep

Useful characters and patterns:

  • ^: Start of line (e.g., ^root)
  • $: End of line (e.g., sh$)
  • .: Any single character (e.g., r..t)
  • [ ]: Character set (e.g., [Pp]attern)
  • [^ ]: Negated set (e.g., [^a-m]att)
  • *: Zero or more repetitions (e.g., oo*)

egrep - Extended grep

egrep supports extended regular expressions, including:

  • +: One or more repetitions (e.g., [a-z]+ark)
  • ?: Zero or one (e.g., unix[0-9]?)
  • x|y: Match x or y (e.g., apple|orange)
  • ( ): Grouping (e.g., search(es|ing)+)

fgrep - Fixed String grep

fgrep searches for fixed strings and does not interpret regular expression characters.

2. Searching Files with find

The find command searches for files and directories based on conditions such as name, type, modification time, size, and permissions.

Syntax:

find path condition [action]

Common conditions:

  • -name: Match file name
  • -type: Match file type (e.g., f = file, d = directory)
  • -mtime / -atime: Match modification/access time
  • -user: Match owner
  • -size: Match file size
  • -newer: Newer than a reference file
  • -perm: Match permissions

Common actions:

  • -print: Display file path (default)
  • -ls: Long listing of results
  • -exec command {} \;: Execute a command on each result
  • -ok command {} \;: Like exec but with confirmation

3. Searching Commands with which

The which command locates executables in the user's PATH environment.

Example:

  • which ls: Shows the path to ls
