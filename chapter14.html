Chapter 14: Basic of System Management

Author: 19100586 unix101

1. System Administrator

  • Manages system operations and security
  • Account: root, home: /root
  • Change user: su [-] [username]

2. Booting and Shutdown

  • shutdown: Shutdown system
    Options: -y (no confirm), -g (delay), -i (init level), now (immediate)

3. User Management

  • useradd: Add user account
    Options: -u, -g, -G, -d, -m, -c
  • usermod: Modify user info
    Options: -u, -g, -d, -s, -l, -m, -c
  • userdel: Delete user account
    Option: -r (remove home)
  • wall: Send message to all users
  • /etc/motd: Login message

4. Group Management

  • groupadd: Add group
    Options: -g (GID), -o
  • groupmod: Modify group
    Options: -g, -o, -n
  • groupdel: Delete group

5. Root Password Management

  • Use complex passwords, change regularly
  • If lost, recover using system tools (e.g. sudo passwd root after reboot)

6. Service Management

  • svcs: View services
    Options: -a, -p, -l, -x
  • svcadm: Manage services
    Subcommands: enable, disable, restart

7. Package Management

  • pkg: Install and manage packages
    Subcommands: install, list, search

8. Network Management

  • ifconfig: View IP config
    Options: -a
  • ipadm: Manage IP addresses
