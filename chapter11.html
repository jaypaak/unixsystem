Chapter 11: Unix Utility

Author: 19100586 unix101

1. File Information: wc

  • wc: Count lines, words, bytes, or characters
    Options:
    • -c: Byte count
    • -m: Character count
    • -l: Line count
    • -w: Word count
    Syntax: wc [options] filename

2. Sorting Files: sort

  • sort: Sort lines of text files
    Options:
    • -b: Ignore leading blanks
    • -r: Reverse order
    • -c: Check if sorted
    • -u: Unique lines
    • -d: Dictionary order
    • -o file: Save result to file
    • -t char: Field separator
    • -k #,#: Sort by fields
    • -n: Numeric sort
    Syntax: sort [options] filename

3. Splitting Files: split

  • split: Split large files into smaller parts
    Options:
    • -b n: Split by n bytes
    • -n: Split by n lines
    Syntax: split [options] [filename]

4. Removing Duplicates: uniq

  • uniq: Filter out repeated lines
    Options:
    • -c: Count occurrences
    • -d: Show duplicated lines
    • -u: Show unique lines
    Syntax: uniq [options] [filename]

5. Cutting Fields: cut

  • cut: Extract parts of lines
    Options:
    • -c list: Select characters
    • -f fields: Select fields
    • -d char: Field delimiter
    Syntax: cut [options] [filename]

6. Pasting Files: paste

  • paste: Merge lines of files
    Options:
    • -s: Serial (one after another)
    • -d char: Field delimiter
    Syntax: paste [options] file1 file2

7. File Dump: dd

  • dd: Convert and copy files
    Options:
    • bs=n: Block size
    • conv=lcase: Convert to lowercase
    • conv=ucase: Convert to uppercase
    Syntax: dd [options] [if=input] [of=output]
