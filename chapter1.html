What is Unix?

The first operating system developed in 1969 at AT&T Bell Labs, and the predecessor of the well-known
"Linux" developed by Linus Torvalds in 1991. Unix has the following six characteristics.

  • Interactive system
  • Multi-user system
  • Multitasking system
  • High portability and scalability
  • Hierarchical tree file system
  • Provides various additional features

Structure of Unix


Kernel: The core of the operating system, responsible for managing computer resources such as processes, memory, devices, and the file system.

Shell: Provides the user interface, receives user commands through this interface, processes them, and outputs the results.

Connecting to Unix

ssh: $ [id]@[Server IP address], set and enter password

telnet: $ telnet [Server IP address], set ID and password

Entering Commands

Command Structure: command -[option] [argument]

Command: banner, man, date...

Option: Specifies detailed functionality of the command, starts with a hyphen and uses numbers, lowercase, or uppercase letters.

Argument: Value passed to the command, usually a file/folder name.

Exiting Unix

  • $ logout
  • $ exit
  • $ ctrl + d
