Chapter 13: Bash Shell Programming

Author: 19100586 unix101

1. Shell Script

  • Shell Script: Program interpreted and executed by the shell
  • Components: Commands, control structures, variables
  • Execution: ./scriptname or bash scriptname
  • Structure:
    • #!: Specify the interpreter (e.g., #!/bin/bash)
    • #: Comments

2. Variables

  • Types: Shell variables, environment variables
  • Assignment: var=value
  • String Manipulation: %, # for pattern removal, length
  • Positional Parameters: $1, $2, etc.
  • Command Substitution: `command` or $(command)

3. Input

  • read: Read input from user
  • here document (<<): Input until a terminator string

4. Operators

  • Types: Arithmetic, comparison, logical, bitwise
  • Usage: let, (( )) for arithmetic

5. Control Structures

  • Conditional Statements: if, case
  • Loops: for, while, until, select
  • Loop Control: continue, break
  • Test Conditions: String tests, file tests, numeric tests

6. Functions

  • Definition: function name { commands; }
  • Calling Functions: Just use the function name
  • Return Values: return [n], stored in $?
  • Delete Function: unset -f functionname

7. Debugging

  • bash -x: Print each command as it runs
  • trap: Handle signals, monitor variables with DEBUG
