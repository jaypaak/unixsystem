Chapter 10: Internet & Transport Commands

Author: 19100586 unix101

1. User Communication: write/wall/talk/mesg

  • write: Send messages to a logged-in user
    Syntax: write user [terminal]
    End message with Ctrl+D
  • wall: Broadcast a message to all users
    Syntax: wall [filename]
    Used for urgent messages; general users must be in tty group
  • talk: Direct conversation with another user
    Syntax: talk user[@host] [terminal]
    Supports 1:1 conversations locally or with remote hosts
  • mesg: Control permission for receiving messages
    Syntax: mesg [y|n]
    'y' allows, 'n' denies messages

2. Sending Emails: mailx

  • mailx: Command-line email tool
    Syntax: mailx [options] [email]
    -s: Specify subject
    Internal commands:
    • h: List mail headers
    • r: Reply to a message
    • d [num]: Delete mail (last read if no num)
    • u num: Undelete mail
    • s filename: Save mail to file
    • q: Quit and save to ~/mbox
    • x: Exit without saving

3. Host and User Info

  • hostname: Show system hostname
  • nslookup domain: Query DNS for domain or IP
    Syntax: nslookup [domain]
  • ping: Check network connectivity
    Syntax: ping host
  • finger: Display user information
    Syntax: finger [user|user@host|@host]

4. File Transfer: ftp

  • ftp: File Transfer Protocol for uploads/downloads
    Syntax: ftp host
    Internal commands:
    • cd: Change server directory
    • lcd: Change local directory
    • ls: List server directory
    • ascii/bin: Set transfer mode
    • get/mget: Download file(s)
    • put/mput: Upload file(s)
    • prompt: Toggle prompt for multiple files
    • hash: Show progress
    • bye: Exit ftp
