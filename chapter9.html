Chapter 09: File Archive & Compression Commands

Author: 19100586 unix101

1. File Archiving with tar

Archive: A collection of files grouped into one for storage.

tar (tape archive) - Common options:

  • c: Create archive
  • t: List archive contents
  • x: Extract archive
  • f: Specify archive file
  • v: Verbose output
  • h: Follow symbolic links

2. Archiving and Compressing with jar

jar (Java Archive Tool)

  • cvf: Create and compress jar archive
  • c0vf: Create archive without compression
  • tvf: View contents
  • xvf: Extract files
  • u: Update archive
  • i: Create index
  • m: Use manifest file
  • 0: No compression

3. .Z Files: compress/uncompress

  • compress: Creates .Z compressed files
  • -f: Force compression
  • -c: Output to screen, no file
  • -v: Show compression info
  • zcat: Display contents of .Z files
  • uncompress: Restore .Z files

4. .gz Files: gzip/gunzip

  • gzip: Compress files into .gz
  • -d: Decompress
  • -l: List contents
  • -r: Recursive compression
  • -t: Test archive
  • -v: Verbose
  • gzcat: View .gz file content
  • gunzip: Decompress .gz files

5. .zip Files: zip/unzip

  • zip: Compress files into .zip
  • -v: Verbose output
  • -r: Recursive compression
  • -u: Update archive with new or changed files
  • unzip: Extract .zip files

6. .bz2 Files: bzip2/bunzip2

  • bzip2: High compression open source tool
  • -d: Decompress
  • -l: Show content
  • -t: Test file
  • -v: Verbose output
  • --best: Max compression
  • bzcat: View contents of .bz2
  • bunzip2: Decompress .bz2 files
