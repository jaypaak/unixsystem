Chapter 12: Understanding of Unix Shell

Author: 19100586 unix101

1. Functions and Types of Unix Shell

  • Shell: Provides user interface
  • Functions:
    • Command interpreter: Interprets and executes commands
    • Programming features: Supports script-based automation
    • User environment setup: Customizes user settings
  • Types of Shell:
    • Bourne Shell (sh)
    • Korn Shell (ksh)
    • C Shell (csh)
    • Bash (bash)
    • Others: tcsh, dtksh, zsh

2. Korn Shell Features

  • Path: /usr/bin/ksh
  • Prompt: $ (user), # (root)
  • Alias: Custom commands saved in .kshrc
  • History: Save and recall command history
  • Filename Autocomplete: Partial name + ESC/TAB
  • Environment Config:
    • System: /etc/profile
    • User: ~/.profile, ~/.kshrc
    • PATH variable, PS1 prompt setup
  • Terminal Settings: stty for key mapping, size, etc.

3. C Shell Features

  • Path: /usr/bin/csh
  • Prompt: % (user), # (root)
  • Alias: Use alias and unalias
  • History: Store and recall command history, set size with set history
  • File Autocomplete: Enable with set filec
  • Environment Config Files:
    • System: /etc/.login
    • User: ~/.login, ~/.cshrc, ~/.logout
    • source filename: Apply changes
