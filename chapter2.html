    Types of Files

  • Regular Files
    • Text (ASCII, vi): source code (c/java), web documents (html)
    • Binary: multimedia, programs, compressed files, encrypted files
  • Special Files
    • Directory Files
    • Device Files
    • Pipe Files

      • Link Files

      • Hard Link: file name is directly connected to data
      • Soft Link (Symbolic): file name is NOT directly connected to data, instead the data block contains the path of the original file (/usr/lib/~)

Directory Hierarchy

  • Home Directory: ~
  • Parent Directory: ..
  • Current Directory: .
  • Subdirectory: /name

Absolute Path and Relative Path

Absolute Path: The location of a specific file based on the home directory
Relative Path: The location of a specific file based on your current position
In the image above, if your current location is /usr, the absolute and relative paths to /bin are:
Absolute Path: /usr/bin
Relative Path: /bin

Other Directory-related Commands

  • pwd: Displays your current location as an absolute path
  • cd [directory]: Move to [directory]
  • ls [options] [directory]: Displays information about files or directories
  • mkdir [directory]: Creates a directory (error if the directory already exists)
  • rmdir [directory]: Deletes a directory
