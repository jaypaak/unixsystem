Chapter 02 : Learn How to use VIM

Author: 19100586 unix101

Mode vs. Non-Mode Type Editors

  • Mode Type
    • Keys are interpreted as commands in command mode.
    • Keys are interpreted as data in insert mode.
    • Special keys are used to switch between modes.
  • Non-Mode Type
    • All keys are considered input data.
    • Commands are issued using combinations of keys (e.g., Ctrl or Alt).
    • Example: Word processors like Word and Excel.

Vim is a mode type editor.

Vim Architecture

  • Buffer: Temporary storage for editing content during insert mode.
  • Disk: Where edited content is saved from the buffer.

Entering Insert Mode

  • i: Insert at cursor position
  • a: Insert after the cursor
  • o: Add a new line below the current line
  • I: Insert at the beginning of the current line
  • A: Insert at the end of the current line
  • O: Add a new line above the current line

Moving the Cursor

  • h: Move left
  • l: Move right
  • j: Move down
  • k: Move up
  • 0 or ^: Move to the first character of the line
  • $: Move to the end of the line

Scrolling the Screen

  • Ctrl + u: Scroll up half a page
  • Ctrl + d: Scroll down half a page
  • Ctrl + b: Scroll up a full page
  • Ctrl + f: Scroll down a full page
  • :line_number: Move to the specified line
  • :0: Move to the first line
  • :$: Move to the last line

Saving and Exiting Vim

  • :q: Quit if no changes were made
  • :q!: Quit without saving changes
  • :w [filename]: Save changes, optionally to a new file
  • :wq / :wq!: Save changes and quit

Editing and Deleting

  • r: Replace character under cursor
  • #r[char]: Replace # characters with given character
  • cw: Change the current word
  • #cw: Change # words
  • s: Replace character(s) starting from cursor
  • #s: Replace # characters starting from cursor
  • x: Delete character under cursor
  • dw: Delete the current word
  • dd: Delete current line
  • #dd: Delete # lines starting from current line
  • u: Undo last action
  • .: Repeat last action

Copy & Paste

  • yy / #yy: Copy the current line / # lines
  • p: Paste below current line
  • P: Paste above current line

Specifying Ranges

  • :1,$ or :%: From first to last line
  • :10,20: From line 10 to 20
  • :#y: Copy line #
  • :range y: Copy lines within range
  • :#d: Delete line #
  • :range d: Delete lines within range
  • :pu: Paste buffer content after current line

Search and Replace

  • /string: Search downward
  • ?string: Search upward
  • n: Next match in same direction
  • N: Next match in reverse direction
  • :range s/old/new/: Replace first match of 'old' with 'new' in range
  • :range s/old/new/g: Replace all matches of 'old' with 'new' in range
  • :%s/old/new/g: Replace all occurrences in the entire file

Other Useful Commands

  • :!shell_command: Run a shell command within Vim
  • :sh: Open shell temporarily, return with exit
